Mysql


    Mysql 数据库及其内部INNODB引擎版本对照表：
    
    Mysql版本           innodb引擎版本
    
    5.1.x                     1.0.x版本(官方称为InnoDB Plugin)
    
    5.5.x                     1.1.x版本
    
    5.6.x                     1.2.x版本
    
    
    
    查询mysql版本命令是：Select @@Version;
    
    查询innodb引擎版本命令是：show variables like 'innodb_version';
