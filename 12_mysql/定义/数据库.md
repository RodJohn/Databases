数据库
在MySQL中，数据库是用于存储和操作诸如表，数据库视图，触发器，存储过程等数据的对象的集合。

数据库操作
查看数据库列表
SHOW DATABASES;
默认有mysql/information_schema等库
创建数据库
CREATE DATABASE 
[IF NOT EXISTS] 
database_name 
[DEFAULT] CHARACTER SET charset_name ;

IF NOT EXISTS是语句的可选子句。 可防止创建同名数据库的错误。
查看数据库创建概述
SHOW CREATE DATABASE  database_name ;
修改数据库字符集
ALTER DATABASE 
[IF EXISTS] 
database_name 
CHARACTER SET charset_name ;
删除数据库
DROP DATABASE
database_name ;
选中数据库
USE database_name ;
查看当前选中的数据库
SELECT DATABASE();
