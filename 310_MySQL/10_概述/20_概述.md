# 发行版

发行版类型

    社区版
        完全开源.免费
        配合开源工具使用
    企业版
        和社区版代码一样
        通过插件提供额外特性
        
    企业版插件
        线程池插件
        审计插件
        SQL安全插件
        官方热备工具
        官方监控平台

版本类型

    GA(Generally Available) 正式版本 官方持续维护
    Archived verson         归档版本 官方不再维护
    Milestone Release       正在开发的版本
    Release Candidate       发布预选版
