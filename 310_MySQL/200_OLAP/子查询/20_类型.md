# 概述

    子查询的结果作为比较条件


# ANY

作用

    ANY一般和比较操作符一起使用
    表示有一个

示例

    where s1 > ANY (1 , 4, 8)
    
    只要(1 , 4, 8)中有一个小于s1,则等式成立

特别

    如上例子中,
    如果子查询结果为空 ()
    或者为(NULL ,NULL)
    则判断为false

SOME  IN

    SOME是ANY的别名
    IN相当于=ANY

# 子查询表

定义

    子查询的结果是一张表(派生表),
    并和普通表一样出现在FROM子句.

特点

    派生表是完全虚拟的
    优化器不清楚派生表的信息,explain很难进行解析
